# Синхронизация локальной папки с Яндекс.Диском

## Описание проекта

Программа обеспечивает одностороннюю синхронизацию содержимого локальной папки с папкой на Яндекс.Диске.  
Все изменения в локальной папке (добавление, изменение, удаление файлов) автоматически применяются к облачному хранилищу.  
Синхронизация работает в режиме реального времени с заданным интервалом.

**Особенности:**

- Поддерживается только синхронизация **локальная → облако** (облачные файлы, отсутствующие локально, удаляются).
- При изменении локального файла он перезаписывается в облаке.
- Логирование всех операций в файл.
- Настраиваемый интервал синхронизации.

## Требования

- Python 3.8 или выше
- Установленная библиотека `requests`
- Токен доступа к Яндекс.Диску (OAuth)

## Установка и настройка

1. **Склонируйте или скопируйте файлы проекта** в отдельную папку:
   - `main.py`
   - `cloud_storage.py`
   - `sync_engine.py`
   - `logger_setup.py`
   - `config.ini` (создайте по шаблону ниже)

2. **Шаблон**:

````
[Settings]
local_folder = C:/MyFolder          # путь к локальной папке
cloud_folder = MyCloudBackup         # имя папки на Яндекс.Диске (будет создана автоматически)
token = y0_AgAAAAA...                # ваш OAuth-токен
sync_interval = 60                   # интервал синхронизации в секундах
log_file = sync.log                  # путь к файлу лога
````
